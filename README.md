## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:
- **Node.js** (version 18 or higher)
- **npm** or **yarn** or **bun** (recommended)
- **Git**

### 🔧 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/shoury-dev/portfolio-website-shouryjeet.git
   cd portfolio-website-shouryjeet/shouryjeet-tech-strategy
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   # or
   bun install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   bun dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:5173](http://localhost:5173)

## 🎨 Customization

### 📝 **Navigation & Routes**
Update routing in `src/App.tsx`:

```typescript
// Add new routes
<Route path="/new-page" element={<NewPage />} />
```

### 💼 **Case Studies**
- Add new case studies in `src/pages/case-studies/`
- Update case studies list in `src/pages/CaseStudies.tsx`
- Follow the existing pattern for consistency

### 🎨 **Styling**
- Global styles: `src/index.css`
- Tailwind config: `tailwind.config.ts`
- Component styles: Use Tailwind classes and shadcn/ui components

### 🏗️ **Components**
- UI components: `src/components/ui/` (shadcn/ui)
- Custom components: `src/components/`
- Use TypeScript for type safety
